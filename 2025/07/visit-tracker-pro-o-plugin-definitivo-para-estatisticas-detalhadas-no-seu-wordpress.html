Visit Tracker Pro: O plugin definitivo para estatísticas detalhadas no seu WordPress

Publicado em 2025-07-09 12:18:10

Se geres um site ou blogue em WordPress, sabes que conhecer o comportamento dos teus visitantes é essencial. Saber que páginas são mais visitadas, quais os IPs mais ativos e quanto tempo os leitores permanecem nos teus artigos pode fazer a diferença entre crescer ou estagnar.

🌟 Apresentamos o Visit Tracker Pro

O Visit Tracker Pro é um plugin WordPress leve e eficaz que regista todas as visitas ao teu site, guardando:

  • Endereço IP
  • Página visitada
  • Agente do navegador
  • Hora da visita
  • Sessão do utilizador

Tudo isto com o objetivo de fornecer estatísticas detalhadas acessíveis diretamente no painel de administração.

🔬 Funcionalidades principais

  1. Ranking de Páginas Mais Visitadas
    Vê rapidamente quais as páginas que estão a atrair mais tráfego.
  2. Tempo Médio de Visita por Página
    Mede o tempo de permanência com base nas mudanças de página por sessão.
  3. Ranking de IPs Mais Ativos (com País)
    Vê quais os IPs com mais visitas e de que país são. Com cache local para evitar sobrecarga.

⚙️ Implementação técnica

  • Cria uma tabela própria wp_vtp_analytics na base de dados.
  • Usa template_redirect para capturar visitas sem interferir no conteúdo.
  • Admin com menu próprio e três tabelas interativas com dados agregados.
  • API externa ip-api.com para identificar país do IP, com cache local em ficheiro JSON.

💸 Benefícios de marketing

  • Identifica quais artigos mantêm mais tempo os leitores.
  • Deteta origem de tráfego suspeito por IP.
  • Otimiza os teus conteúdos com base em interesse real dos visitantes.

✅ Pronto a usar

Basta instalar o ficheiro ZIP como qualquer plugin WordPress. Não requer configuração inicial.

A partir do painel "Visit Tracker", terás acesso imediato às tuas estatísticas internas.

Desperta a mente analítica do teu WordPress.
