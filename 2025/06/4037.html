Publicado em 2025-06-28 18:10:46

🛡️

No mundo digital actual, em que os ataques automatizados e acessos abusivos são uma constante ameaça à integridade dos websites, nasce uma solução portuguesa robusta e inteligente: Vigilant Guardian.

Este plugin foi cuidadosamente concebido para atuar como um vigia silencioso, sempre atento à atividade suspeita, pronto a bloquear abusos e a notificar o administrador com precisão e clareza.

🔍 O que faz o Vigilant Guardian?

1. Detecção de Acessos Abusivos em Tempo Real
O plugin monitoriza continuamente os acessos por endereço IP. Se um IP fizer mais de 40 pedidos por minuto, é automaticamente considerado abusivo e entra na lista negra.

2. Bloqueio Automático com Persistência
O IP é bloqueado instantaneamente e guardado numa tabela MySQL (wp_vigilant_guardian_blocked_ips), garantindo que futuros acessos desse IP sejam recusados sem comprometer o desempenho.

3. Notificações Imediatas com Geolocalização
Quando um IP é bloqueado, o administrador recebe uma notificação NTFY, com detalhes como país, cidade, e motivo do bloqueio. Tudo enviado para o canal seguro:
http://ntfy.com/Alarms

4. Exclusão Inteligente do IP do Administrador
Através de um sistema automático, o plugin detecta o IP público do administrador e impede que ele seja bloqueado acidentalmente — uma proteção essencial para quem gere o sistema.

5. Painel de Controlo Completo
No painel de administração do WordPress, o Vigilant Guardian disponibiliza:

  • Listagem de IPs bloqueados
  • Opção para remover bloqueios manualmente
  • Logs detalhados com data, IP, localização e motivo do bloqueio
  • Gráficos diários de ataques para análise visual da atividade maliciosa

🌍 Segurança com Inteligência e Elegância

O Vigilant Guardian não é apenas um plugin: é um escudo inteligente, capaz de aprender com os padrões de tráfego e agir antes que o ataque afete o teu site. É compatível com servidores sob proxy reverso (X-Forwarded-For) e adapta-se a ambientes exigentes.

📦 Instalação

  1. Acede ao teu WordPress > Plugins > Adicionar Novo
  2. Carrega o ficheiro .zip do plugin Vigilant Guardian
  3. Ativa e configura no menu lateral: “Vigilant Guardian”
  4. Está pronto! O teu site está agora sob vigilância constante.

🔧 Requisitos

  • WordPress 5.5 ou superior
  • PHP 7.4 ou superior
  • Acesso à base de dados MySQL
  • Ligação ao serviço NTFY para notificações

📈 Futuras Funcionalidades (em desenvolvimento)

  • Machine Learning para deteção de padrões novos de ataque
  • Integração com reCAPTCHA para desafios em tempo real
  • Partilha de IPs maliciosos entre instalações (rede de proteção distribuída)

Vigilant Guardian representa a união entre simplicidade, poder e vigilância. Ideal para quem deseja uma solução automática, transparente e eficaz contra abusos e acessos maliciosos.
